底座選擇及比較

底座是整體浴室的靈魂，底座對了，浴室就在對的基礎上

為了給您更堅固耐用的產品，我們一直研發、提升，不只超越業界，更勝過傳統工法

不袗底座是EZSHOWER的專利，我們創新而不將就，注重每一個零件與細節

　

EZSHOWER 第四代底座：鋁合金骨架 + 硬殼防水板 + 不袗水盤 + 面材

 

最堅固耐用的衛浴底座，踩起來的硬度和泥作相同，但比泥作還堅固，用鐵鎚敲也不會破

承重達500公斤以上，採面的支撐，確保不凹陷變形

EZSHOWER 第三代底座：鋁合金支架 + 硬殼防水板 + 不袗水盤 + 面材

 

堅固耐用的底座，鋁合金支架強度高，可以作整體浴室，承重250公斤以上

　

EZSHOWER 第二代底座：塑鋼支架 + 硬殼防水板 + 不袗水盤 + 面材

 

實用CP值高的底座，採用塑鋼支架，可以作整體淋浴房，承重150公斤以上

　

第一代底座：塑鋼支架 + 硬殼防水板 + 不袗水盤 + 面材

 

一般市面上用的整體浴室底座，價格便宜操作方便

第1代產品 EZSHOWER2015年推出第1代淋浴房，顛覆傳統工法，提供更簡單的建置衛浴方式
第2代產品 將第1代產品大部分功能(如多段噴頭、音響)拿掉，只保留基本淋浴功能，操作更簡單，故障機率更低
第3代產品 板材換成插接式結構，組裝、搬運更方便，可以加掛配件。且更容易維護，不移動底座就可以快速找到進水管線
第4代產品 強化結構，採用12支台灣生產粗壯骨架，1支骨架重量就比前幾代整座骨架重。因結構超堅固，所以可搭配各式門片
第5代產品 研發新型專利底座並在台灣生產，不用打矽利康就可以完全防水，更多板材及配件選擇，可建置高質感整體浴室
第6代產品 底座再升級，不袗與磁磚完美結合，踩踏質感與傳統衛浴相同，防水卻更好且可以打開底座輕鬆維護排水系統
